#  Аналіз пасажирських перевезень АТ "Укрзалізниця"

##  Про проєкт
Цей проєкт присвячений дослідженню відкритих даних про кількість перевезених пасажирів на постійних приміських маршрутах. Ми аналізуємо розподіл пасажиропотоку за регіональними філіями, щоб виявити закономірності та навантаження на залізничну мережу.

**Джерело даних:** [Портал відкритих даних (data.gov.ua)](https://data.gov.ua/)  
**Набір даних:** Кількість перевезених пасажирів на постійних приміських маршрутах (з розподілом за регіональними філіями).

---

##  Гіпотези для дослідження
1. **Регіональне лідерство:** Південно-Західна та Львівська залізниці мають стабільно вищий пасажиропотік порівняно з іншими філіями через густоту населення.
2. **Сезонність:** Кількість пасажирів має чітко виражені піки у літні місяці та грудень (період відпусток та свят).
3. **Якість звітів:** Дані містять аномальні викиди або нульові значення в періоди активних змін графіків руху.
4. Статистичний аналіз філій (змінено в гілці Research).
---

##  Структура проєкту
Проєкт організований за модульним принципом для забезпечення автоматизації (CI/CD):

* `src/data_load/` — автоматичне завантаження та первинна обробка файлів Excel/CSV.
* `src/data_quality/` — перевірка даних на цілісність, наявність дублікатів та пропусків.
* `src/data_research/` — розрахунок статистичних показників та агрегація даних.
* `src/visualization/` — генерація аналітичних графіків та звітів.
