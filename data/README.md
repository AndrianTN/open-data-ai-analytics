# Дані проєкту

Ця директорія призначена для зберігання даних, які використовуються в проєкті.

## Структура

- `raw/` — сирі (необроблені) дані. Ця папка не зберігається в Git, оскільки файли можуть бути великими.
- `processed/` — оброблені або очищені дані.
- `README.md` — опис структури даних.

## Джерело даних

У проєкті використовуються відкриті дані для аналітичного дослідження.

Файли необхідно завантажити локально та помістити в папку `data/raw/`.

## Примітка

Великі файли даних не додаються в репозиторій.  
У Git зберігається лише код, документація та результати аналізу.

## Запуск проєкту через Docker

Проєкт складається з кількох сервісів:

- `data_load` — завантаження або підготовка даних;
- `data_quality` — перевірка якості даних;
- `data_research` — аналіз даних;
- `visualization` — побудова графіків.
